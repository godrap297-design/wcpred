# GitHub Pages Deployment Guide

## Deploy

1. Create a GitHub repository.
2. Upload every file in this project folder.
3. Open repository Settings.
4. Open Pages.
5. Choose Deploy from a branch.
6. Select main and /root.
7. Save.

GitHub Pages publishes the website at:

https://YOUR_USERNAME.github.io/YOUR_REPOSITORY/

## Notes

- No build step is required.
- Keep index.html in the repository root.
- Firebase web credentials can be public when Firestore rules are configured correctly.
- Never commit private service account keys.

## Pre-Launch Test

1. Open the public URL.
2. Toggle dark mode.
3. Complete a prediction.
4. Confirm duplicate phone protection.
5. Open admin/index.html.
6. Enter actual winners and verify scores.
7. Download CSV.
